Projects

Todo-App

I created a simple To-Do app using HTML, CSS, and JavaScript that allows users to add, complete, and delete tasks. One of its key features is that tasks are saved in the browser using localStorage, so they remain even after the page is reloaded. The app provides a clean and minimal interface, making it easy to manage daily tasks effectively.

Blackjack Game

I built a browser-based Blackjack game using HTML, CSS, and JavaScript that simulates the classic card game experience. Players can draw cards, calculate the sum, and aim to hit 21 without going over. The game includes basic logic for win/lose conditions, a user-friendly interface, and real-time score updates. It’s a fun and interactive way to demonstrate JavaScript fundamentals like conditionals, loops, and event handling.

Zomato UI Clone

I created a responsive clone of Zomato’s user interface using only HTML and CSS, focusing on recreating the look and feel of the original food delivery platform. The project includes sections like the navigation bar, search bar, featured restaurants, and food categories. It helped me strengthen my skills in layout design, Flexbox, and responsive styling while replicating a real-world UI.
